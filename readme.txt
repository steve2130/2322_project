# Readme

This program is a multi-threaded web server that can response with HTML, JPG and PNG file with corresponding header upon receiving HTTP request.



To run this program, please make sure that you have the latest version of Python installed.
Python can be downloaded here: https://www.python.org/

Double click on "main.py" to launch the program.

Type "localhost:2130" on the url bar in your browser. A demo website will be displayed on the browser.




"test.py" is for testing the program with differnet header and HTTP method

"logging.py" is for tracking user requests and server response status.
